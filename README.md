## Демо
url: https://practise-project-jet.vercel.app/ (vercel)
## Стек технологий
### Frontend
- React
- TypeScript
- Vite
- SCSS Modules 
- Redux Toolkit
- Yup
- Axios
- React Hook Form
- React Router
### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL
- JWT
- bcrypt

### Database

- PostgreSQL (Neon)

# Запуск проекта

## Запуск через Docker

### Требования

- Docker Desktop

### Запуск

```bash
git clone https://github.com/cench-dev/practise-project.git

cd practise-project

docker compose up --build
```

После запуска:

Backend:

```
http://localhost:5000
```

Для запуска frontend в режиме разработки:

```bash
npm install
npm run dev
```

Frontend будет доступен по адресу:

```
http://localhost:5173
```
